# Trading Bot - Binance Futures Testnet

## Setup

1. Install dependencies:
   pip install -r requirements.txt

2. Add API keys in .env file

3. Run:
   python cli.py

## Features
- Market Orders
- Limit Orders
- BUY / SELL
- Logging
- Error Handling

## Example
Symbol: BTCUSDT
Side: BUY
Type: MARKET
Quantity: 0.01
